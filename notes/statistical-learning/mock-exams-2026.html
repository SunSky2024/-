模拟卷 A

题型结构：惩罚回归 15 分，分类计算 10 分，选择题 15 分，SVM/核方法 10 分，判断题 12 分，因果识别 12 分，双稳健 14 分，纵向 IPTW 12 分。

1. 简答与推导：Ridge、Lasso、Elastic Net （15 分）

  1. 简述 Ridge、Lasso、Elastic Net 的目标函数及主要区别。
  2. 设 \(X^\top X=I\)，推导正交设计下 Lasso 的估计形式。
  3. Lasso 估计是否无偏？为什么？

2. 分类指标计算 （10 分）

某二分类数据集中正例和负例各 100 个。分类器总体准确率为 82%，其中正例被错判为负例的比例为 12%。

  1. 求灵敏度（sensitivity）。
  2. 求负例被正确判断的比例，即特异度（specificity）。
  3. 如果把阈值调低，一般灵敏度和特异度如何变化？

3. 选择题 （15 分，每小题 3 分）

  1. 增大正则化强度通常会使偏差、方差如何变化？
    1. 偏差减小，方差减小
    2. 偏差增大，方差减小
    3. 偏差减小，方差增大
    4. 偏差增大，方差增大
  2. 以下哪种做法通常不能缓解过拟合？
    1. 增加样本量
    2. 减少变量个数
    3. 增加更高阶的多项式项
    4. 增加 \(L_2\) 正则化
  3. 在 kNN 中，想得到更平滑的分类边界，应当：
    1. 增大 \(k\)
    2. 减小 \(k\)
    3. 去掉所有训练样本
    4. 只保留离群点
  4. 不可约误差主要来自：
    1. 模型没有调参
    2. 训练样本太少
    3. 结局中的随机噪声或未观测因素
    4. 变量选择太多
  5. 下列哪项最接近随机森林中 OOB 样本的用途？
    1. 强制每棵树完全相同
    2. 估计泛化误差和变量重要性
    3. 让训练误差恒为 0
    4. 替代所有正则化方法

4. SVM 与核技巧 （10 分）

  1. SVM 可以用于线性不可分的数据吗？需要引入什么？
  2. SVM 最终分类超平面是否只与支持向量有关？简要说明。
  3. 简述核技巧的作用。

5. 判断题：树、随机森林与非参数方法 （12 分，每小题 3 分）

判断正误，并给出一句理由。

  1. 如果 \(X_1\) 与 \(Y\) 边际独立，则 \(X_1\) 一定不会出现在树的切分中。
  2. 单棵树不剪枝时通常更容易过拟合。
  3. 随机森林是许多树预测结果的平均或投票，因此主要可以降低方差。
  4. KDE 的带宽越小，估计一定越好。

6. 因果图、G 公式与 IPTW （12 分）

判断识别 \(A\) 对 \(Y\) 的总效应时是否需要控制 \(L\)，并说明理由。

L ---> A ---> Y L ----------> Y
  1. 写出 \(ACE=E(Y^1)-E(Y^0)\) 的三个常用识别假设。
  2. 写出 G 公式。
  3. 写出点处理情形下的 IPTW 估计思想。

7. 双重稳健估计 （14 分）

给定

$$ \hat\psi_{DR,1} = \frac1n\sum_{i=1}^n \left[ \frac{A_i}{\hat\pi(L_i)} \{Y_i-\hat m_1(L_i)\} +\hat m_1(L_i) \right]. $$
  1. 说明 \(\pi(L)\) 和 \(m_1(L)\) 分别表示什么。
  2. 证明只要 \(\hat\pi(L)\) 或 \(\hat m_1(L)\) 有一个正确，该估计量就一致估计 \(E(Y^1)\)。
  3. 双重稳健是否意味着两个模型都可以错？

8. 纵向 IPTW 与删失 （12 分）

  1. 写出两时间点处理 \((A_0,A_1)\) 下 sequential exchangeability 的直观含义。
  2. 简述使用 IPTW 拟合均值 MSM 的流程。
  3. 为什么出现失访/删失时还需要逆删失概率权重？

模拟卷 B

本套卷与 A、C 平级：选择题和判断题占比相同，证明题仍以双重稳健为核心，但问法换成另一种常见形式。

1. 简答与推导：线性模型和惩罚回归 （15 分）

  1. 写出普通最小二乘的正规方程，并说明什么时候 \((X^\top X)^{-1}\) 不存在。
  2. 设 \(X^\top X=I\)，推导正交设计下 Ridge 的估计形式。
  3. 为什么 Lasso 更适合做变量选择，而 Ridge 通常不会把系数压成精确的 0？

2. 分类指标计算 （10 分）

某模型把所有样本都判为正例。数据中正例 60 个，负例 40 个。

  1. 求灵敏度、特异度和准确率。
  2. 逻辑回归可以用于分类问题吗？它直接输出的是类别还是概率？
  3. 若类别极不平衡，为什么只看准确率可能误导？

3. 选择题 （15 分，每小题 3 分）

  1. BIC 相比 AIC 通常：
    1. 惩罚更弱，更偏向复杂模型
    2. 惩罚更强，更偏向简单模型
    3. 完全不惩罚模型复杂度
    4. 只适用于分类树
  2. QDA 相比 LDA 的主要区别是：
    1. QDA 假设各类均值相同
    2. QDA 允许不同类别有不同协方差矩阵
    3. QDA 不能用于分类
    4. QDA 不需要训练数据
  3. 下列哪项最能体现核技巧？
    1. 显式计算所有高维特征再回归
    2. 只通过 \(K(x_i,x_j)\) 计算高维内积
    3. 完全不使用训练样本
    4. 把所有核函数都设为 0
  4. 在点处理因果推断中，正值性假设大致要求：
    1. 每个协变量层内都有接受和不接受处理的可能
    2. 所有结局都必须相同
    3. 处理必须连续
    4. 协变量不能影响处理
  5. 纵向 IPTW 中，稳定权重的分母通常是：
    1. 边际处理概率
    2. 给定历史协变量和处理历史的处理概率
    3. 结局的无条件均值
    4. 训练误差

4. RKHS、KRR 与 SVM （10 分）

  1. 解释 RKHS 中“再生性质”的含义。
  2. 表示定理说明核岭回归的解可以写成什么形式？
  3. 比较 hinge loss 与平方损失在分类问题中的角色。

5. 判断题：树、随机森林与核平滑 （12 分，每小题 3 分）

判断正误，并给出一句理由。

  1. 分类树中 Gini、entropy 和 misclassification error 都可衡量节点不纯度。
  2. Bagging 对高度不稳定的学习器通常更有帮助。
  3. 随机森林中 \(mtry\) 的作用之一是降低树与树之间的相关性。
  4. 核回归中带宽越大，估计越局部，方差越大。

6. 因果图、混杂与碰撞点 （12 分）

分别判断识别 \(A\) 对 \(Y\) 的总效应时是否应控制 \(X\)，并说明理由。

(1) X ---> A ---> Y X ----------> Y
(2) A ---> X ---> Y
(3) A ---> X <--- Y
  1. 指出 \(X\) 在三个图中分别扮演什么角色。
  2. 哪一种情况下控制 \(X\) 可能引入偏差？

7. 双重稳健估计 （14 分）

点处理下，\(E(Y^1)-E(Y^0)\) 的双重稳健估计量可写成两个 \(E(Y^a)\) 的双重稳健估计之差。

  1. 写出 \(E(Y^0)\) 对应的双重稳健估计量形式。
  2. 证明当处理模型 \(P(A=0\mid L)\) 正确时，即使结局模型错设，估计仍一致。
  3. 证明当结局模型 \(E(Y\mid A=0,L)\) 正确时，即使处理模型错设，估计仍一致。

8. 纵向 IPTW 与删失计算 （12 分）

某个体两时间点处理概率估计为：

  • \(\hat P(A_0=a_0)=0.50,\quad \hat P(A_0=a_0\mid L_0)=0.25\)；
  • \(\hat P(A_1=a_1\mid A_0=a_0)=0.60,\quad \hat P(A_1=a_1\mid A_0=a_0,L_0,L_1)=0.30\)；
  • \(\hat P(C_0=0\mid L_0,A_0)=0.80,\quad \hat P(C_1=0\mid \bar L_1,\bar A_1,C_0=0)=0.75\)。
  1. 计算稳定处理权重。
  2. 计算逆删失概率权重。
  3. 写出合并权重，并说明其用途。

模拟卷 C

本套卷仍与 A、B 平级，不再做“提升卷”。重点是换数据、换图、换表述，检验是否真正理解而不是记住题面。

1. 简答与推导：模型选择与惩罚项 （15 分）

  1. 简述训练误差和测试误差为什么会随模型复杂度出现不同变化。
  2. 写出 Ridge 和 Lasso 的约束形式，并用几何直觉解释 Lasso 为什么更容易得到稀疏解。
  3. SCAD/MCP 相比 Lasso 想解决什么问题？

2. 分类指标计算 （10 分）

某二分类模型的混淆矩阵如下：TP=45，FN=15，FP=20，TN=120。

  1. 计算灵敏度、特异度和准确率。
  2. 如果研究目标更重视“不漏掉正例”，应更关注灵敏度还是特异度？
  3. ROC 曲线横轴和纵轴分别是什么？

3. 选择题 （15 分，每小题 3 分）

  1. 在 kNN 回归中，\(k=1\) 通常对应：
    1. 高偏差、低方差
    2. 低偏差、高方差
    3. 高偏差、高方差
    4. 不可约误差为 0
  2. Lasso 的主要特点之一是：
    1. 一定比 Ridge 无偏
    2. 可以产生精确零系数
    3. 不能用于高维数据
    4. 没有调参参数
  3. 随机森林每次切分只考虑部分变量，主要目的是：
    1. 增加树之间的相关性
    2. 降低树之间的相关性
    3. 让每棵树变成线性模型
    4. 避免使用 bootstrap
  4. 若 \(A\to X\to Y\)，识别 \(A\) 对 \(Y\) 的总效应时控制 \(X\) 会：
    1. 控制混杂
    2. 控制中介，通常改变总效应目标
    3. 一定提高总效应估计效率且无副作用
    4. 没有任何影响
  5. 删失权重的直观作用是：
    1. 把删失者的结局补成 0
    2. 调整因失访导致的选择偏差
    3. 删除所有未删失者
    4. 替代处理权重

4. SVM、核函数与核密度估计 （10 分）

  1. 写出软间隔 SVM 中松弛变量的含义。
  2. 高斯核为什么可以得到非线性分类边界？
  3. KDE 中带宽过大和过小分别会造成什么现象？

5. 判断题：线性模型、分类与树模型 （12 分，每小题 3 分）

判断正误，并给出一句理由。

  1. 增加变量个数一定会降低测试误差。
  2. LDA 假设各类具有相同协方差矩阵，QDA 不要求这一点。
  3. 树模型可以用于回归，也可以用于分类。
  4. 随机森林中的 OOB 样本是每棵树 bootstrap 抽样时未被抽到的样本。

6. 因果识别、G-computation 与 IPTW （12 分）

  1. 给出一致性假设 \(Y=Y^A\) 的含义。
  2. 比较 G-computation 和 IPTW：一个主要建模什么，另一个主要建模什么？
  3. 在随机试验中，为什么调整强预测 \(Y\) 的基线协变量 \(L\) 可能降低方差？

7. 双重稳健估计 （14 分）

考虑

$$ \hat\psi_{DR} = \frac1n\sum_{i=1}^n \left[ \frac{A_i}{\hat\pi(L_i)} \{Y_i-\hat m_1(L_i)\} +\hat m_1(L_i) - \frac{1-A_i}{1-\hat\pi(L_i)} \{Y_i-\hat m_0(L_i)\} -\hat m_0(L_i) \right]. $$
  1. 指出这是估计哪个因果参数。
  2. 说明 \(\hat m_1,\hat m_0,\hat\pi\) 分别是什么模型。
  3. 给出双重稳健证明的两种情况。

8. 纵向 MSM、IPTW 流程与删失 （12 分）

  1. 为什么时间变化混杂会使普通回归调整变得困难？
  2. 写出 IPTW procedure 的三个主要步骤。
  3. 若删失依赖于既往协变量和处理历史，说明用逆删失概率权重需要什么假设。

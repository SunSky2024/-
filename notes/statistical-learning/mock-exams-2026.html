模拟卷 A

本卷偏“概念辨析 + 小计算 + 点处理因果证明”。题目覆盖面广，但每问都控制在历年回忆卷的短答难度附近。

1. 选择题 （20 分，每小题 2 分）

  1. 在 kNN 回归中，\(k\) 增大通常会：
    1. 使估计更粗糙，方差增大
    2. 使估计更平滑，方差减小
    3. 使不可约误差消失
    4. 使训练误差必定增大到 1
  2. 岭回归最直接的作用是：
    1. 对系数做收缩
    2. 删除所有非零变量
    3. 改变响应变量定义
    4. 保证训练误差为 0
  3. Lasso 相比 Ridge 更容易产生稀疏解，主要因为：
    1. \(\ell_1\) 约束区域有尖角
    2. \(\ell_2\) 约束区域有尖角
    3. Lasso 没有调参参数
    4. Lasso 不使用损失函数
  4. 逻辑回归直接建模的是：
    1. \(P(Y=1\mid X)\)
    2. \(P(X\mid Y=1)\)
    3. \(E(X)\)
    4. 样本距离
  5. 软间隔 SVM 引入松弛变量主要是为了：
    1. 处理线性不可分或允许少量违反间隔
    2. 取消支持向量
    3. 使所有样本都在间隔线上
    4. 把分类问题变成聚类问题
  6. KDE 带宽太小通常导致：
    1. 估计过度平滑
    2. 估计很抖，方差较大
    3. 所有密度值相等
    4. 无法归一化
  7. 随机森林中随机选择候选变量的目的之一是：
    1. 降低树之间相关性
    2. 让所有树完全相同
    3. 不再需要 bootstrap
    4. 保证每棵树不分裂
  8. 一致性假设 \(Y=Y^A\) 的意思是：
    1. 实际接受的处理对应的潜在结果等于观测结果
    2. 所有人的潜在结果都相同
    3. 处理与协变量独立
    4. 不存在删失
  9. IPTW 主要依赖建模：
    1. 处理分配概率
    2. 核函数形状
    3. 树的最大深度
    4. 岭回归自由度
  10. 纵向删失权重用于：
    1. 调整失访造成的选择偏差
    2. 把删失者的结局强制设为 0
    3. 替代所有处理权重
    4. 让样本量变大

2. 短计算：分类指标与树切分 （12 分）

  1. 某二分类模型有 TP=36、FN=14、FP=20、TN=80。计算灵敏度、特异度和准确率。
  2. 某节点中正例 24 个、负例 16 个。切分后左节点正例 18 个、负例 2 个，右节点正例 6 个、负例 14 个。计算父节点与两个子节点的 Gini 指数，并说明切分是否降低不纯度。

3. 判断题 （12 分，每小题 3 分）

判断正误，并给出一句理由。

  1. 训练误差下降并不保证测试误差下降。
  2. QDA 允许不同类别有不同协方差矩阵。
  3. Bagging 的主要作用之一是降低不稳定学习器的方差。
  4. 控制碰撞点一定能减少混杂偏差。

4. 线性模型与惩罚方法 （12 分）

  1. 写出线性回归的正规方程，并说明 \(X^\top X\) 不可逆时会出现什么问题。
  2. 比较 Ridge、Lasso 与 Elastic Net 在“变量选择”和“相关变量处理”上的差异。
  3. 为什么 SCAD/MCP 被称为试图减少估计偏差的惩罚方法？

5. 核方法与 SVM （10 分）

  1. 写出核岭回归中表示定理给出的函数形式。
  2. 解释 SVM 中支持向量为什么关键。
  3. 简述核技巧为什么能处理非线性边界。

6. 因果图与识别 （12 分）

考虑以下三个 DAG。判断识别 \(A\) 对 \(Y\) 的总效应时是否应该控制 \(X\)，并说明理由。

(a) X ---> A ---> Y X ----------> Y
(b) A ---> X ---> Y
(c) A ---> X <--- Y
  1. 分别指出 \(X\) 是混杂变量、中介变量还是碰撞点。
  2. 写出点处理 \(ACE=E(Y^1)-E(Y^0)\) 的三条常用识别假设。

7. 双重稳健证明思路 （12 分）

设 \(\pi(L)=P(A=1\mid L)\)，\(m_1(L)=E(Y\mid A=1,L)\)。考虑

$$ \hat\psi_1=\frac1n\sum_{i=1}^n\left[ \frac{A_i}{\hat\pi(L_i)}\{Y_i-\hat m_1(L_i)\} +\hat m_1(L_i) \right]. $$
  1. 说明该估计量估计的目标量。
  2. 分别说明处理模型正确、结局模型正确时为什么仍一致。

8. 纵向 IPTW 与删失 （10 分）

  1. 写出两时间点处理问题中 sequential exchangeability 的直观含义。
  2. 简述用 IPTW 拟合均值 MSM 的步骤。
  3. 如果删失依赖既往协变量，逆删失概率权重需要什么假设？

模拟卷 B

本卷换成“方法选择 + 局部计算 + 纵向权重计算”。题型难度与 A 持平，但不沿用同一知识点顺序。

1. 选择题 （20 分，每小题 2 分）

  1. 若 \(p\) 接近或超过 \(n\)，普通最小二乘最直接的问题是：
    1. \(X^\top X\) 可能不可逆或估计不稳定
    2. 所有变量都自动无关
    3. 训练误差必定等于测试误差
    4. 不需要正则化
  2. BIC 相比 AIC 通常：
    1. 对模型复杂度惩罚更强
    2. 完全不惩罚复杂度
    3. 只能用于 kNN
    4. 总是选择最大模型
  3. 朴素贝叶斯的“朴素”主要指：
    1. 给定类别后各特征条件独立
    2. 所有类别概率相等
    3. 不用训练数据
    4. 只能处理连续变量
  4. ROC 曲线纵轴通常是：
    1. 灵敏度
    2. 特异度
    3. 训练误差
    4. 模型自由度
  5. 硬间隔 SVM 要求：
    1. 训练样本线性可分
    2. 所有样本权重相同
    3. 没有支持向量
    4. 不需要约束
  6. Nadaraya-Watson 估计量本质上是：
    1. 核加权局部平均
    2. 全局线性分类器
    3. 代价复杂度剪枝
    4. 后门路径准则
  7. 随机森林中 OOB 样本是：
    1. 某棵树 bootstrap 抽样时未被抽到的样本
    2. 所有树都必须使用的样本
    3. 只用于计算训练误差的样本
    4. 被删除的异常值
  8. 正值性假设被严重违反时，IPTW 常见问题是：
    1. 权重极端，估计不稳定
    2. 所有权重都等于 1
    3. 不再需要协变量
    4. 结局模型自动正确
  9. 直接标准化主要建模：
    1. 结局回归 \(E(Y\mid A,L)\)
    2. 训练样本距离
    3. 树的随机切分
    4. 核函数积分
  10. 边际结构模型中的“边际”强调：
    1. 目标是处理方案对总体潜在结局均值的模型
    2. 只研究协变量分布
    3. 只适用于无处理变量问题
    4. 只能用普通最小二乘直接调整所有 \(L_t\)

2. 局部计算：kNN 与核回归 （12 分）

  1. 测试点附近 5 个样本的类别为 \(1,0,1,1,0\)。写出 \(k=1,3,5\) 时的 kNN 分类结果。
  2. 写出 Nadaraya-Watson 核回归估计量。若带宽变大，局部性、偏差和方差通常如何变化？

3. 判断题 （12 分，每小题 3 分）

判断正误，并给出一句理由。

  1. Lasso 一定比 Ridge 有更小的预测误差。
  2. 局部线性回归可以缓解一部分边界偏差。
  3. 随机森林只能用于回归，不能用于分类。
  4. 如果 \(A\to X\to Y\)，控制 \(X\) 会改变“总效应”的目标。

4. 惩罚回归与模型选择 （12 分）

  1. 写出 Ridge 的惩罚形式和约束形式。
  2. 说明有效自由度随 Ridge 参数 \(\lambda\) 增大如何变化。
  3. 简述交叉验证选择调参参数的基本步骤。

5. 分类与判别分析 （10 分）

  1. 写出贝叶斯分类规则。
  2. 比较逻辑回归、LDA、QDA 的决策边界特点。
  3. 解释 masking problem 的基本含义。

6. 树与随机森林 （12 分）

  1. 解释代价复杂度剪枝中“拟合误差”和“树复杂度惩罚”的权衡。
  2. 为什么 bootstrap 后每棵树大约有一部分样本没有被抽到？这些样本有什么用途？
  3. 变量重要性的 OOB permutation 思想是什么？

7. 因果估计量辨析 （12 分）

  1. 写出 \(E(Y^a)\) 的 G 公式。
  2. 写出点处理下 \(E(Y^1)\) 的 IPTW 估计量。
  3. 如果结局模型正确但处理模型错误，双重稳健估计量为什么仍可一致？请给出证明思路。

8. 纵向权重计算与删失 （10 分）

某个体有两次处理，估计概率为：

  • \(\hat P(A_0=a_0)=0.6,\quad \hat P(A_0=a_0\mid L_0)=0.3\)；
  • \(\hat P(A_1=a_1\mid A_0=a_0)=0.5,\quad \hat P(A_1=a_1\mid A_0=a_0,L_0,L_1)=0.25\)；
  • \(\hat P(C_0=0\mid L_0,A_0)=0.8,\quad \hat P(C_1=0\mid \bar L_1,\bar A_1,C_0=0)=0.5\)。
  1. 计算稳定处理权重。
  2. 计算逆删失概率权重。
  3. 说明最终分析权重如何合并。

模拟卷 C

本卷偏“混合选择 + 方法解释 + DAG 判断”。与 A、B 同级，但减少对单一历年热点的依赖。

1. 选择题 （20 分，每小题 2 分）

  1. 测试误差的偏差-方差分解中，不可约误差通常：
    1. 无法通过换模型完全消除
    2. 等于训练误差
    3. 只由 Ridge 产生
    4. 只在分类问题中存在
  2. Elastic Net 同时使用：
    1. \(\ell_1\) 与 \(\ell_2\) 惩罚
    2. Gini 与 entropy
    3. IPTW 与删失权重
    4. ROC 与 AUC
  3. 多分类 LDA 中 Fisher 判别坐标的目的之一是：
    1. 寻找有利于类别分离的低维方向
    2. 强制所有类别均值相等
    3. 取消协方差矩阵
    4. 把监督学习变成无监督学习
  4. 支持向量通常指：
    1. 对分类边界起决定作用的训练样本
    2. 所有测试样本
    3. 被随机森林丢弃的变量
    4. 核函数的积分常数
  5. 表示定理的核心含义是：
    1. 无限维函数优化可化为有限个核函数的线性组合
    2. 所有函数都必须是线性的
    3. 核函数不能用于回归
    4. 正则化项必须为 0
  6. 分类树中节点越纯，Gini 指数通常：
    1. 越小
    2. 越大
    3. 恒等于 1
    4. 与类别比例无关
  7. DAG 中后门路径的起点方向通常是：
    1. 从 \(A\) 指向 \(A\) 的父节点方向，即进入 \(A\) 的箭头
    2. 从 \(A\) 指向 \(Y\) 的直接箭头
    3. 任何通过中介的路径
    4. 所有被阻断的路径
  8. 双重稳健估计“稳健”的含义是：
    1. 处理模型或结局模型有一个正确即可一致
    2. 两个模型都错也必然一致
    3. 小样本一定无偏
    4. 不需要正值性
  9. 时间变化混杂的困难在于某些 \(L_t\) 可能：
    1. 既影响后续处理，又受既往处理影响
    2. 完全独立于处理和结局
    3. 只能是连续变量
    4. 不能出现在 DAG 中
  10. 逆删失概率权重适用的直观前提是：
    1. 给定已观测历史后，是否删失可视为随机
    2. 删失一定与所有变量无关
    3. 删失者的结局都为 0
    4. 删失概率不能被建模

2. 小计算：ROC 与 kNN （12 分）

  1. 某模型 TP=50、FN=10、FP=30、TN=110。计算灵敏度、特异度和 false positive rate。
  2. 某点最近 7 个邻居的类别为 \(0,1,1,0,1,0,1\)。分别给出 \(k=3\) 与 \(k=7\) 的分类结果。

3. 判断题 （12 分，每小题 3 分）

判断正误，并给出一句理由。

  1. 增加正则化强度通常会降低模型复杂度。
  2. 朴素贝叶斯要求所有特征在边际上独立。
  3. 剪枝可以看作对树复杂度的一种控制。
  4. MSM 的参数通常直接解释为给定当前协变量后的条件效应。

4. 非参数方法与树模型 （12 分）

  1. 比较 kNN 回归与核回归：二者的局部性分别由什么控制？
  2. 解释边界偏差和曲率偏差的来源。
  3. 随机森林为什么通常比单棵树稳定？

5. 惩罚分类与核方法 （10 分）

  1. 写出 hinge loss 的形式，并说明哪些样本会产生正损失。
  2. 高斯核为什么可以产生非常灵活的非线性边界？
  3. 核岭回归与普通岭回归的联系是什么？

6. 点处理因果推断 （12 分）

  1. 写出 \(E(Y^a)\) 在无未测量混杂和正值性下的识别公式。
  2. 比较 G-computation 与 IPTW：一个主要估计什么，一个主要估计什么？
  3. 随机试验中调整强预测结局的基线协变量为什么可能提高效率？

7. DAG 与 d 分离 （12 分）

考虑图：

L ---> A ---> Y L ---> M ---> Y A ---> M
  1. 识别 \(A\) 对 \(Y\) 的总效应时，是否应控制 \(M\)？为什么？
  2. 是否应控制 \(L\)？为什么？
  3. 说明“总效应”和“直接效应”在调整变量选择上的区别。

8. 双重稳健与纵向 IPTW （10 分）

  1. 写出 \(E(Y^1)-E(Y^0)\) 的双重稳健估计量由哪两类模型组成。
  2. 简述双重稳健证明的两条路线。
  3. 纵向 IPTW 中，处理权重和删失权重为什么通常要相乘进入最终加权分析？
